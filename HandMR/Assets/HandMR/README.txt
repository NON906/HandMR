# HandMR

This is assets for hands tracking and 6DoF head tracking with Dangla(https://dangla.jp/) MR or smartphone VR (cardboard etc).
This works Android (ARM64 and ARCore) or iOS (ARKit).
By using this asset, you can easily create an MR / VR / AR app with hand tracking.

## How to use

1. Import this assets.
2. The setting window will open, so execute contents.
3. Open sample scenes or placement Assets/HandMR/Prefabs/HandMRManager.prefab on scene.
